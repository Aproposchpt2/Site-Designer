Features

Built for businesses. Ready for agencies.

🌐

Embeddable Anywhere

Drop the Intake Engine on any website as a standalone page or embedded form. Works with any CMS, landing page builder, or custom site.

🧠

AI Categorization

Each submission is automatically categorized by request type, urgency, and suggested next action — powered by the Claude API.

⚡

Instant Dashboard Sync

Every submission appears on the Lead Manager dashboard in real time via Supabase. No manual import. No delay.

📧

Email + SMS Alert

The business owner receives an instant notification with the full record summary — including urgency level and suggested follow-up action.

🏷️

Industry Templates

Pre-configured intake templates for home services, legal, dental, real estate, and more. Matching urgency language and request types per industry.

🔒

TCPA Compliant

SMS consent fields, opt-in language, and privacy disclosures built in. Every intake record captures consent timestamp and text for compliance.
